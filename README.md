## 💡 技術的なこだわり・アピールポイント

本作はアクションゲームとしての「ラリーの爽快感」と「手ごたえのある敵AI」を両立するため、単なるエンジン依存ではなく、**C++による数学的アプローチ（ベクトル演算・物理逆算）**をコアシステムに組み込んでいます。

### 1. 空間図形を用いた「未来予測AI」による先回り移動
* **【課題】** 球速が上がると、ボールの「現在Y座標」を追従するだけの単純なAIでは移動が追いつかず、ラリーが成立しない問題がありました。
* **【解決】** ボールの「現在位置と速度ベクトル（半直線）」と「敵の移動可能ライン（線分）」の交点を算出する独自関数 `GetLineSegmentAndRayIntersection` をC++で実装しました。
* **【結果】** ボールが到達する前に「予測地点（未来）」へ先回りするプロアクティブなAIを構築し、プレイヤーの高速ショットにも対応できる、手応えのある敵AIを実現しています。

### 2. 物理演算の逆算による「手触り」の向上（動的重力制御とドライブ回転）
* **【課題】** 標準の物理演算のまま速い球を打つと、放物線を描いて敵コート外へオーバーしてしまい、ゲームとしての爽快感が損なわれる問題がありました。
* **【解決】** ターゲットとなる着弾点から逆算し、到達に必要なZ軸の初速を動的に計算して代入するロジックを実装しました。
* **【結果】** さらに、球速（ターゲットへの到達時間の短さ）に応じてボールにかかる重力倍率を動的に高めるアルゴリズムを組み込みました。これにより、ネットすれすれを通過してコートに鋭く突き刺さる「ドライブ回転（トップスピン）」を数学的に再現し、アクションゲームとしての最高の手触りを実現しました。

## 🎮 操作方法 (Xboxコントローラー推奨)

本作はゲームパッドでのプレイを想定して直感的に操作できるよう調整されています。

| アクション | 入力ボタン (Xbox準拠) |
| :--- | :--- |
| **移動・選択** | 左スティック |
| **返球（打つ方向の指定）** | 右スティックを倒す |
| **サーブ** | Y ボタン |
| **決定・スキル使用** | A ボタン |

## 🚀 インストールと実行方法

※ 本作の実行には Windows 環境が必要です。

1. GitHubの **[Releases]** ページから `Windows Build v1.0` (zipファイル) をダウンロードします。
2. ダウンロードした zip ファイルを任意のフォルダに展開します。
3. 展開したフォルダ内にある実行ファイル（`.exe`）を起動するとゲームが開始されます。
